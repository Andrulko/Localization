安裝擴充功能非常重要，因為單獨程式是無法做任何事的。

# 安裝
1. [點此](https://premid.app/downloads) 前往我們的下載頁面
2. 選擇 **FireFox**
3. 搜尋 `about:addons`
4. 點擊右上角的小齒輪圖示
5. 選擇 **從檔案安裝擴充套件**
6. 選擇你最近下載的檔案
7. Accept the installation by clicking on **Add**.
