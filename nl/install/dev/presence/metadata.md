# Metadata.json

Als u een presence in de winkel wilt publiceren en deze wilt laden via de extensie, dan moet u de `metadata.js` bestand in uw `presence.js` map aanmaken.

Het voorbeeld van dat bestand kan hieronder worden gevonden.

```javascript
{
  "author": {
    "name": "GEBRUIKER",
    "id": "ID"
  },
  "contributors": [{
    "name": "GEBRUIKER",
    "id": "ID"
  }],
  "service": "SERVICE",
  "description": {
    "en": "BESCHRIJVING"
  },
  "url": "URL",
  "regExp": "REGEXP",
  "iFrameRegExp": "REGEXP",
  "version": "VERSIE",
  "logo": "URL",
  "thumbnail": "URL", "URL",
  "color": "#45A8FC",
  "tags": ["CATEGORY", "TAG"],
  "category": "CATEGORIE",
  "iframe": false
}
```
