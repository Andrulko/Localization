# Metadata.json

Wenn Sie eine Präsenz im Store veröffentlichen und über die Erweiterung laden möchten, sollten Sie die Datei `metadata.json` in Ihrem Ordner `presence.js` erstellen.

Das Beispiel für diese Datei finden Sie unten.

```javascript
{
  "author": {
    "name": "USER",
    "id": "ID"
  },
  "contributors": [{
    "name": "USER",
    "id": "ID "
  }],
  " service ":" SERVICE ",
  " description ": {
    " de ":" DESCRIPTION "
  },
  " url ":" URL ",
  " regExp ":" REGEXP ",
  "iFrameRegExp": "REGEXP",
  "version": "VERSION",
  "logo": "URL",
  "thumbnail": "URL",
  "color": "# 45A8FC",
  "tags": ["CATEGORY", "TAG"],
  "category": "CATEGORY",
  "iframe": false
}
```
