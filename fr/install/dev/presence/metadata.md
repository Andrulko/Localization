# Métadonnées.json

Si vous voulez publier une présence dans le magasin et la charger via l'extension, vous devez créer les métadonnées `. son` fichier dans votre dossier `presence.js`.

L'exemple de ce fichier peut être trouvé ci-dessous.

```javascript
{
  "author": {
    "name": "USER",
    "id": "ID"
  },
  "contributeurs": [{
    "name": "UTILISATEUR",
    "id": "ID"
  }],
  "service": "SERVICE",
  "description": {
    "fr": "DESCRIPTION"
  },
  "url": "URL",
  "regExp": "REGEXP",
  "iFrameRegExp": "REGEXP",
  "version": "VERSION",
  "logo": "URL",
  "thumbnail": "URL",
  "color": "#45A8FC",
  "tags": ["CATÉGORIE", "TAG"],
  "category": "CATÉGORY",
  "iframe": false
}
```
