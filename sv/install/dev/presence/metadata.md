# Metadata.json

Om du vill publicera en närvaro i butiken och ladda den via tillägget bör du skapa `metadata. son` -filen i mappen `presence.js`.

Exemplet med den filen kan hittas nedan.

```javascript
{
  "författare": {
    "namn": "ANVÄNDAR",
    "id": "ID"
  },
  "bidragsgivare": [{
    "namn": "ANVÄNDAR",
    "id": "ID"
  }],
  "service": "TJÄNSTE",
  "description": {
    "sv": "DESCRIPTION"
  },
  "url": "URL",
  "regExp": "REGEXP",
  "iFrameRegExp": "REGEXP",
  "version": "VERSION",
  "logo": "URL",
  "thumbnail": "URL",
  "color": "#45A8FC",
  "taggar": ["CATEGORY", "TAG"],
  "category": "CATEGORY",
  "iframe": false
}
```
